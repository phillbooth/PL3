# Design

## Design Summary

Describe the visual style and user experience.

## Pages / Screens

| Page | Purpose |
|---|---|
| Home | Main landing page |
| Dashboard | Main user area |
| Settings | User or admin settings |

## User Flow

1. User opens the app.
2. User completes the main action.
3. App validates the input.
4. App returns a result or error.

## Layout Rules

- Mobile-first design
- Clear navigation
- Simple forms
- Consistent spacing
- Accessible colours
- Readable text sizes

## Components

- Header
- Footer
- Navigation
- Buttons
- Forms
- Alerts
- Cards
- Tables

## Accessibility Notes

- Use clear labels.
- Use readable contrast.
- Avoid relying only on colour.
- Ensure keyboard navigation where relevant.
